The server requires JDK >= 1.5

Windows
-------

1. cd genkey
2. build the server using: buildAll.bat
3. cd ..
4. launch the server by double clicking on serve.jar
5. in your browser, visit <http://localhost:8080/>

Unix
----

1. cd genkey
2. build the server using: ./buildAll.sh
3. cd ..
4. launch the server with: java -jar serve.jar
5. in your browser, visit <http://localhost:8080/>
