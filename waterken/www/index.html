WaterkenTM server is installed!

You have successfully installed the WaterkenTM server. Server documentation is available online.

Things to do…

  • Follow the instructions to configure an SSL key pair
  • Follow the instructions to edit your homepage
  • Setup the Bang Tutorial on your new installation
    1. create the server-side vat with the command:

      java -jar spawn.jar example org.waterken.bang.Main test

    2. copy the URL emitted by the previous command
    3. start the server with the command:

      java -jar serve.jar

    4. follow the directions in the Bang Tutorial, using the copied URL in place of the online test page
